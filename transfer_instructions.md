# מדריך להעברת הפרויקט למחשב אחר

כדי להמשיך לעבוד על "עין צופיה" ממחשב אחר, בצע את השלבים הבאים:

## שלב 1: העתקת הקבצים
1. גש לתיקייה במחשב הנוכחי:
   `c:\Users\aharo\.gemini\antigravity\scratch\ein-tsofia-auth`
2. העתק את **כל התיקייה** לכונן חיצוני (USB) או שירות ענן (Google Drive).
   > **שים לב:** חשוב מאוד שקובץ `.env.local` יעבור גם הוא! הוא מכיל את המפתחות ל-Firebase.

## שלב 2: הפעלה במחשב החדש
1. העתק את התיקייה למחשב החדש.
2. פתח את התיקייה באמצעות התוכנה (VS Code / Antigravity).
3. פתח טרמינל (Terminal) בתוך התיקייה.
4. אם לא העתקת את תיקיית `node_modules` (התיקייה הכבדה), הרץ פקודה זו כדי להתקין רכיבים:
   ```bash
   npm install
   ```
5. להפעלת המערכת, הרץ:
   ```bash
   npm run dev
   ```
6. פתח דפדפן בכתובת: `http://localhost:3000`

## המשך עבודה על EMMA
התחלנו לתכנן את EMMA (העוזרת החכמה). הקבצים שיצרנו (התמונה והתוכנית) נמצאים בתוך התיקייה, כך שתוכל להמשיך בדיוק מאיפה שעצרנו.
